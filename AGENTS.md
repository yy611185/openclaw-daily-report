# 多因子AI美股投研系统 - Agent 工作流程

## 执行指令

当用户说"生成今日报告"、"daily report" 或类似指令时，按以下 **严格顺序** 执行：

### Phase 1：数据采集
1. 调用 `us-daily-market-data` 技能，获取指数、宏观、板块数据
2. 调用 `us-daily-news-brief` 技能，获取新闻与情绪数据
3. 确认两项数据均获取成功后进入下一阶段

### Phase 2：因子分析与选股
4. 调用 `us-daily-equity-research` 技能，执行多因子分析
5. 计算基本面因子（收入增长、盈利变化、估值）
6. 计算技术面因子（MA、RSI、支撑阻力）
7. 结合情绪面数据（来自 Phase 1 的新闻分析）
8. 综合打分，选出 Top 5 股票

### Phase 3：报告生成
9. 调用 `us-daily-report-writer` 技能，整合所有数据生成报告
10. 确保报告包含全部 7 个章节

### Phase 4：自检循环
11. 执行自检（Critic Mode），检查数据真实性、逻辑一致性、完整性
12. 如发现问题，执行纠错（Fix Mode）并重新验证
13. 最多循环 2 次
14. 输出最终状态：FINAL APPROVED 或 CONDITIONAL APPROVED

## 重要规则

- **不得跳过任何 Phase**
- **不得在数据未获取时就开始分析**
- **不得在分析未完成时就生成报告**
- **自检是强制步骤，不可省略**
- **每日报告格式必须一致**
